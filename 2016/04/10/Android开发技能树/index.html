发表于 2016-04-10   |   分类于 Android   |  

#Android开发技能树
本文的目的是为Android/Java初学者所提供的一个简单版开发技能树，不涉及进阶知识，仅为开发一个Android App所需的最主要知识。

开发工具

  1. JDK和Android SDK 基础开发环境
  2. Android Studio 开发IDE
  3. eclipse 阅读其他项目源码时可能需要，了解基本功能即可
  4. VScode/Notepad++ 阅读简单代码
  5. Genymotion 更快的模拟器

辅助工具

  1. MOU/MarkdownPad 书写文档使用
  2. Sketch 原型图及素材制作
  3. git 版本控制工具（基本命令、gitflow）
  4. shadowsocks 爬墙工具

java知识点

  1. 基本语法
  2. oo语法（引用/继承/多态/接口/内部类/匿名内部类/回调）
  3. 线程（线程通信/同步/多线程/线程池）
  4. 容器与泛型
  5. 反射与注解
  6. 异常(声明/捕获)

Android基本知识点

  1. 四大组件生命周期及通信（Activity、Service、ContentProvider、BroadcastReceiver）
  2. UI布局（LinearLayout、RelativeLayout、FrameLayout）
  3. 基础控件与自定义控件（Listview、Recyclerview、ViewPager、View、ViewGroup）
  4. 数据绑定（Adpter）
  5. 多线程（AsyncTask、Handler、Rxjava）
  6. 数据持久化存储（SqLite、SharedPreferences、Internal Storage）
  7. Http与Parse（OkHttp和Gson，JSON与XML解析）
  8. WebViews（js与Native的交互）
  9. 资源文件（color、string、drawable、style、dimens）
  10. Material Design （设计规范与开源控件）

Android其它相关内容

  1. 构建工具（Gradle的使用）
  2. 打包（keystore）
  3. 混淆（ProGuard）
  4. 自动化测试（robotium/Appium）

Android第三方服务

  1. 推送（LeanCloud/极光）
  2. 分享（ShareSDK）
  3. 数据统计（友盟）
  4. 短信服务（LeanCloud/阿里大鱼）
  5. 质量跟踪（bugly/BugHD ）
  6. 内测平台（fir.im）
  7. 加固平台（360加固）
  8. IM（融云）

开发架构

  1. 设计模式
  2. MVC/MVP/MVVM
  3. 基类（BaseActivity）、通用函数（utils）、常量（config/constant）

##性能优化

  1. 内存管理（内存抖动/图片等资源的加载/内存泄漏）
  2. 网络优化（缓存机制/超时参数/请求合并/gzip）
  3. 布局及渲染优化（过度重绘/嵌套过多）
  4. 线程控制（耗时操作不在UI线程/延时操作/加载资源提前操作）
  5. 资源优化（资源的分辨率与格式）
  6. 通用操作优化（数据结构/算法/数据库/逻辑）
